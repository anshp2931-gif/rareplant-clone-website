# 🌿 RarePlant Clone Website

A simple and responsive **RarePlant-inspired website** built using **HTML, CSS, and basic assets**. This project replicates a plant e-commerce UI with multiple pages like home, deals, login, and order tracking.

---

## 🚀 Features

* 🌱 Landing page with hero section
* 💸 Deals/offers page
* 🔐 Login page UI
* 📦 Order status tracking page
* 🎨 Clean and minimal design
* 📱 Responsive layout (if styled accordingly)

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Assets:** Images
